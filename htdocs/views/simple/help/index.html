Prerequisites

... or the author says requirements:
Secure connection (HTTPS)
is required to secure your data (files, account password) and to enable cookies ...
Cookies
are required for settings (language selection, ...), bookmarks, clipboard (copy, cut, paste), folder history,...
JavaScript
enables all features and without JavaScript nothing works in the web interface.

Navigation

The web interface has 3 main parts:
  1. head area at the top with a language switch, date and time, and logon information,
  2. application bar at the left side,
  3. content area with a toolbar, the file and folder list, and an optional status bar

Buttons and icons

Not every button has a label and therefore there are tooltips. Place the mouse cursor over it or put the keyboard focus on it and wait a couple of seconds to get a tooltip. Some tooltips have a help button to get a context help.

Drag and drop support

Drag and drop makes it easier to upload files and folders, or to customize the file and folder list.

Context menu

The file and folder list has a context menu (right mouse lick on a file or folder row) with some standard file actions and a lot of additional features (depending on your local installation).

Toolbar

  • The toolbar changes it's contents depending on the current file or folder selection.
  • At the left side of the toolbar the quick navigation helps you to navigation back to upper folders.
  • Right of the quick navigation are buttons to create files and folders, uploading files and folders, and so on.
  • If no file or folder is selected, you'll find a bookmarks menu
    and a autorefresh menu
    .

Basic tasks

.. with files and folders:
Download a single file
  • single click on the filename,
  • double click on the matching file list row,
  • click on the file actions button
    and select "Download",
  • or open the context menu with a single right click on the matching file list row and select "Download"
Upload a file
  • click on the new menu button
    , select
    and a file dialog opens,
  • or open your favourite file manager and use drag & drop to put your files and folders into the file and folder list
Change to another folder
  • click on the foldername,
  • double click on the matching folder row,
  • use the quick navigation in the toolbar,
  • or use web browser history back or forward
Delete a single file or folder
click on the file actions button
and select "Delete"
(De)select file(s) or folder(s) for further actions
  • single left click in the row (Attention: don't click on the filename, action button, or icon),
  • after one file or folder was (de)selected you can combine the single left click with
    ⇧ Shift
    to (de)select all files and folders between the first and second selected file or folder
Delete a list of files or folders
  1. select a list of files and folders
  2. and use one of the following options:
    • toolbar button
      ,
    • context menu (right click to row) and click on "Delete"

Customization

... of the web interface.

Web interface

Take a look at the
Settings dialog to change the layout and behavoir the web interface.
Show/Hide application bar or header
click on the toggle buttons on the bottom left of the application bar or the bottom right of the head area to (un)collapse or hide one or both

File list

Column order
single left click on column header and hold mouse key down (Drag & Drop)
Column width
  • double left click on table header changes the width in two steps: 1. minimum size, 2. default size,
  • or a single left click on left or right border on column header and hold mouse key down (mouse cursors should changed to a resize cursor before left click)
File and folder list sort order
  • single left click on table header,
  • or use the list options dialog: click on list options icon
    at the right of the table header
Show or hide columns
click on list options icon
at the right of the table header to open the list options dialog
